Function Reference: loglinv

statistics: x = loglinv (p)
statistics: x = loglinv (p, alpha)
statistics: x = loglinv (p, alpha, beta)

Inverse of the log-logistic cumulative distribution function (iCDF).

For each element of p, compute the quantile (the inverse of the CDF) at p of the log-logistic distribution with scale parameter alpha and shape parameter beta. The size of x is the common size of p, alpha, and beta. A scalar input functions as a constant matrix of the same size as the other inputs.

Both parameters, α and β, must be positive reals and p is supported in the range [0,1], otherwise NaN is returned. By default, alpha = 1 and beta = 1.

Further information about the log-logistic distribution can be found at https://en.wikipedia.org/wiki/Log-logistic_distribution

MATLAB compatibility: MATLAB uses an alternative parameterization given by the pair μ, s, i.e. mu and scale, in analogy with the logistic distribution. Their relation to the alpha and beta parameters is given below:

  • alpha = exp (mu)
  • beta = 1 / scale
